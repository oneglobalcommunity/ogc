## Getting Started

1. **Open the website**: Simply open `index.html` in your web browser
2. **No server required**: This is a static website that runs locally
3. **Cross-browser compatible**: Works on Chrome, Firefox, Safari, and Edge

## Customization

### Colors
The website uses CSS custom properties (variables) for easy color customization. Edit the `:root` section in `styles.css`:

```css
:root {
    --primary-red: #CD2C2E;      /* Traditional Korean red */
    --primary-blue: #0047A0;     /* Traditional Korean blue */
    --gold: #FFD700;             /* Gold accent */
    /* ... other colors */
}
```

### Content
- **Restaurants**: Update restaurant information in the restaurants section
- **Events**: Modify event dates and descriptions
- **Contact**: Update contact information and social media links
- **About**: Customize the mission statement and goals

### Images
Currently using Font Awesome icons. To add real images:
1. Create an `images/` folder
2. Add your images
3. Replace `<i class="fas fa-..."></i>` with `<img src="images/your-image.jpg" alt="Description">`
